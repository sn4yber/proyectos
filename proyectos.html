Proyecto: Sistema de Inventario

Este proyecto es un sistema de inventario desarrollado para gestionar productos y existencias de manera eficiente. Permite registrar, actualizar y consultar información de inventario en tiempo real, facilitando el control y la organización de recursos en una empresa o negocio.

El sistema fue diseñado con una interfaz intuitiva y funcionalidades clave como búsqueda, filtrado y reportes automáticos. Es una solución escalable que puede adaptarse a diferentes necesidades empresariales.
